### Introducing Variables

Swift-Primer

**var and let**

```swift
// value can be changed
var str = "Hello, playground"
```

```swift
// value can't be changed 
let otherStr = "Hi, Mer"
```

**Strings**

* Concatenation

```swift
let name = "Mer"
// concatenation
print("Hello " + name)
```

**Integer (whole number)**

```swift
let myInt = 8
```

* Put integer into string

```swift
var age = 35
// Put integer into string
print("My age is \(age)")
```

**Doubles and Floats**

```swift
var a: Double = 8.73
var b: Float = 8.73
var c = 7.12
```

**Boolean**

```swift
let gameOver = false
```
